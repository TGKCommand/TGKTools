Odoo Import Format

The exported CSV is formatted for Odoo's Inventory → Replenishment → Reorder Rules import. Columns match Odoo's expected field names for orderpoint rules.

Export columns: Internal Reference (SKU) · Product · Route · Min Qty · Max Qty · Qty Multiple · UOM · Lead Days
Import Instructions — Odoo
  1. In Odoo, go to Inventory → Configuration → Reorder Rules
  2. Click Import (top menu)
  3. Upload the exported CSV file
  4. Map columns if Odoo prompts — they should auto-match
  5. Click Import to apply rules
⚠ Existing reorder rules for the same SKU+location will be overwritten. Export a backup from Odoo first if needed.
Global Defaults

These apply to all SKUs unless overridden per-row in the calculator.

Calculation Method

Which sales average to use when calculating Min/Max. Can be overridden per SKU.

Min Qty = Effective Weekly Avg × (Lead Time weeks + Min Weeks Cover)
Max Qty = Effective Weekly Avg × (Lead Time weeks + Target Weeks Cover)
QTY to Order = max(Max Qty − Current On Hand, MOQ)
Lead Time weeks = Lead Time days ÷ 7
